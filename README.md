AtLilyPond
==========

LilyPond syntax highlighting in Atom.

**NOTE:** AtLilyPond is an [Atom](http://atom.io) port of my [SubLilyPond](https://www.github.com/yrammos/SubLilyPond) highlighter for Sublime Text. For screenshots, release history, and other pertinent information, please follow the link above.
